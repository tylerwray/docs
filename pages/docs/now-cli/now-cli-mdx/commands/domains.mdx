## Domains

The `now domains` command is used to manage domains under an account, providing functionality to list, inspect, add, remove, purchase, move, transfer-in, and verify domains.

<Note type="warning">
  This command is for <b>advanced use only</b>. You can manage domains with
  further options and greater control under a project's Domains tab from the
  ZEIT Dashboard.
</Note>

### Basic Usage

<Snippet dark text="now domains ls" />
<Caption>Using the <InlineCode>now domains</InlineCode> command to list all domains under an account.</Caption>

<Snippet dark text="now domains rm [domain]" />
<Caption>Using the <InlineCode>now domains</InlineCode> command to remove a domain from an account.</Caption>

<Snippet dark text="now domains buy [domain]" />

<Snippet dark text="now domains move [domain] [account-name]" />

<Snippet dark text="now domains transfer-in [domain]" />

<Snippet dark text="now domains verify [domain]" />
<Caption>Using the <InlineCode>now domains</InlineCode> command to verify a domain for an account.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now domains` command:

- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--scope`
- `--token`

For more information on global options and their usage, refer to the [options section](#options).

## Inspect

The `now inspect` command is used to retrieve information about a deployment referenced either by its unique deployment URL, staging URL, or production domain.

### Basic Usage

<Snippet dark text="now inspect [unique-deployment-url]" />
<Caption>Using the <InlineCode>now inspect</InlineCode> command to retrieve information about a specific deployment.</Caption>

### Extended Usage

<Snippet dark text="now inspect [production-deployment-url]" />
<Caption>Using the <InlineCode>now inspect</InlineCode> command to retrieve information about a production deployment.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now inspect` command:

- `--debug`
- `--global-config`
- `--help`
- `--local-config`
- `--scope`
- `--token`

For more information on global options and their usage, refer to the [options section](#options).
